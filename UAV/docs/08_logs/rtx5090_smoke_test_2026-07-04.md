### 5.2 第一次尝试: bs=2 → OOM

```bash
python src/training/train_sft.py \
  --config configs/rtx5090.yaml \
  --data_dir /root/autodl-tmp/data/smoke_test
```

```
torch.OutOfMemoryError: CUDA out of memory. Tried to allocate 6.75 GiB.
GPU 0 has 31.36 GiB total, 25.17 GiB already in use, only 6.18 GiB free.
```

**根因**: bs=2 × seq_len=3456 的 CE logits 从 bf16 转 fp32 后膨胀到 6.75GB，超出剩余显存。

### 5.3 第二次尝试: bs=1 → 通过

修改 `configs/rtx5090.yaml`:
```yaml
sft:
  per_device_batch_size: 1          # 2 → 1
  gradient_accumulation_steps: 16   # 8 → 16 (有效 batch 仍为 16)
```

```bash
python src/training/train_sft.py \
  --config configs/rtx5090.yaml \
  --data_dir /root/autodl-tmp/data/smoke_test
```

**结果**: ✅ 3 epochs 完成

## 8. VRAM 实测结论

### 8.1 SFT (bs=1, seq=3456)

| 组件 | 占用 |
|------|------|
| 4-bit 权重 + LoRA + Embed | ~5.5 GB |
| Optimizer (Adam m/v) | ~5 GB |
| Activations (gc 后) | ~6 GB |
| CE fp32 中间量 (bs=1) | ~3.5 GB |
| CUDA overhead | ~3 GB |
| **峰值** | **~25 GB / 32 GB** |

bs=2 时 CE fp32 翻倍到 ~7 GB → 超 32GB → OOM。

### 8.2 DPO (bs=1, seq=1536, mu=0)

| 组件 | 占用 |
|------|------|
| Train 模型 (4-bit + LoRA + embed) | ~10 GB |
| Ref 模型 (4-bit 冻结) | ~5 GB |
| 4× forward activations | ~3 GB |
| DPO log-prob 计算 | ~3 GB |
| Backward (4× logits) | ~7 GB |
| CUDA overhead | ~2 GB |
| **峰值** | **~30.4 GB / 32 GB** |

### 8.3 关键结论

| 参数组合 | SFT | DPO |
|------|:--:|:--:|
| bs=2, seq=3456 | ❌ OOM | N/A |
| bs=1, seq=3456 | ✅ 通过 | ❌ OOM |
| bs=1, seq=2048 | N/A | ❌ OOM |
| bs=1, seq=1536, mu=0 | N/A | ✅ 通过 |

**DPO 在 RTX 5090 32GB 上的最佳配置: seq_len=1536 + mu=0**

### 当前 `configs/rtx5090.yaml` 生效参数

```yaml
hardware:
  gpu: "RTX 5090"
  vram_gb: 32
  use_4bit: true

training:
  sft:
    epochs: 3
    per_device_batch_size: 1           # bs=2 → OOM
    gradient_accumulation_steps: 16    # 有效 batch = 16
    max_seq_length: 3456
  dpo:
    epochs: 2
    per_device_batch_size: 1
    gradient_accumulation_steps: 16
    max_seq_length: 1536              # 降级到 1536 才能过
    mu: 0.0                            # 关 SFT anchor 省显存

model:
  backbone: "/root/autodl-tmp/huggingface/models/gemma-3-4b-it"
  torch_dtype: "bfloat16"
  attn_implementation: "sdpa"
```
